DivPeek

CSS animation triggers with jQuery

DivPeek simply watches the divs you specify to see if they are scrolled in to view. If they are, a class is added so you can easily create a css-transform animation without having to mess with the JS. A class is also added when the element is out of view, so for the sake of progressive enhancement you should create an non-viewed animation start state.

DivPeek comes with a handy page overview thumbnail where you can see your current viewport and what's coming up to test you triggers. Oh, and resizing your browser window updates the viewport and (if necessary) triggers animations for state changes.
